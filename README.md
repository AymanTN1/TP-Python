Jeu de devinette (français)
============================

Petit jeu en ligne de commande écrit en Python. Le programme demande un intervalle
au joueur, choisit un nombre secret et le joueur doit deviner ce nombre. Le jeu
indique si la proposition est trop petite, trop grande ou correcte.

Fichiers:
- `guessing_game.py` : script principal du jeu.

Exécution:

Ouvrez un terminal PowerShell et lancez :

```powershell
python .\guessing_game.py
```

Nécessite Python 3.x (recommandé 3.8+). Aucune dépendance externe.
